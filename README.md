# Astrix UI

AstrixUI is a set of ready-to-go React components and hooks to minimize your effort building UI applications.

## Features
- Available for React v18
- Styled using Tailwindcss
- ES6+ and CommonJS support
- TypeScript support

## Installation

```bash
npm i astrix-ui
```
